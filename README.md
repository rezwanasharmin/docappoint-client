# DocAppoint — Doctor Appointment Manager

Live site: **https://docappoint.vercel.app**

DocAppoint is a full-stack doctor appointment booking platform built with Next.js, MongoDB, and Better Auth.

## Features
- Browse top-rated and all available doctors with search & sort
- Secure JWT/session auth (email + Google) via Better Auth
- Book, update, and cancel appointments from a personal dashboard
- Leave reviews after appointments and edit your profile in-app
- Fully responsive UI with light/dark theme toggle and SEO metadata on every page

## Tech
Next.js 14 (App Router) · MongoDB Atlas · Better Auth · Tailwind CSS · DaisyUI · Swiper · react-hot-toast
